### World Bank MCP server

Этот скрипт запускает модуль `:mcp:worldBankServer`, который предоставляет MCP-инструмент `worldbank_list_countries`. Инструмент обращается к открытому API Мирового банка и возвращает список стран с фильтрами по региону и уровню дохода.

#### Запуск

```bash
./gradlew :mcp:worldBankServer:installDist   # один раз
./mcp/world-bank-server/run-world-bank-server.sh
```

Никаких токенов или дополнительных параметров не требуется — API работает без авторизации. При необходимости можно установить переменные окружения:

- `WORLD_BANK_MCP_SCRIPT` — путь до этого скрипта, чтобы переопределить значение по умолчанию в приложении.

#### Использование в приложении

1. Убедитесь, что `MCP_SERVER_SCRIPT` указывает на `mcp/world-bank-server/run-world-bank-server.sh` (либо выставите `WORLD_BANK_MCP_SCRIPT`).
2. При вопросах в духе «Какие страны с высоким доходом?» клиент вызовет MCP tool и вернёт список стран прямо в чате.
