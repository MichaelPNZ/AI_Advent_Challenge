# Командный ассистент

Ассистент, который знает проект (RAG) и умеет работать с сервисами команды через MCP: создаёт задачи, отвечает о статусе, предлагает приоритеты.

## Быстрый старт

1. **Соберите MCP серверы (задачи и документы):**
   ```bash
   ./gradlew :mcp:reminderServer:installDist
   ./gradlew :mcp:docPipelineServer:installDist
   ```
2. **Запустите десктоп-приложение:**
   ```bash
   ./gradlew :desktopApp:run
   ```
3. **Индексируйте проект/FAQ для RAG (UI → «Индексировать»):**
   - Папка: `docs/support-faq` или корень проекта/доков.
4. **Создайте чат и выберите роль «Командный ассистент».**
   - MCP инструменты по умолчанию включают задачи (`reminder_*`), документы (`doc-pipeline`), git, дневные сводки.

## Что умеет

- **Задачи:** `reminder_add_task`, `reminder_list_tasks`, `reminder_complete_task`, `reminder_summary`.
- **Документы:** `doc-pipeline` (поиск/сводка), RAG по индексу.
- **Статусы:** спрашивает summary и списки задач, может уточнить по приоритету/сроку.
- **Рекомендации:** предлагает порядок выполнения на основе приоритета (high > normal > low), сроков и влияния.

## Примеры

- «Создай задачу high: починить авторизацию до пятницы»
- «Покажи задачи с приоритетом high и предложи, что делать первым»
- «Найди в документации, как настроить RAG, и составь чек-лист»
- «Кратко, что поменялось за сегодня?»

## Полезные пути

- Тикеты/задачи: `~/.ai_advent/reminder_tasks.json`
- Индекс RAG: `~/.ai_advent/embedding_index/index.jsonl` (UI) или `~/.ai_advent/support_index` (CLI из support ассистента)
